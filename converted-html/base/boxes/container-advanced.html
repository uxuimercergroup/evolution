Advanced Containers are built out using the Basic Container framework. Normally they contain base information along with customizable options.

Containers in general are used to highlight specific areas of content. The Advanced Container features 3 components - header, body and footer. These components can be used in any combination - all are not required. Options for this container are rounded corners and drop shadows.

Components

  • The Advanced Container is comprised of the following components:
    • Border
    • Header
    • Header Background Color
    • Content
    • Rounded Corner (optional)
    • Drop Shadow (optional)
  • The border color (#6b90b8) should always be a solid, 1 pixel gray line.
  • The headers for Advanced Containers can be designated as either h1, h2, h3, or h4.
  • The Header Background color is #EEEEEE.

Sizing and Layout

  • The padding inside a Advanced Container's body section is 10 pixels on the top, left, and right. There is no padding on the bottom, as this is taken care of by the content inside.
  • Advanced Container height is variable based on content height. Width is determined by the grid where the Advanced Container resides.

Grids

  • Advanced Containers work within the Mercer Evolution Grid System.
  • Advanced Containers can contain grids inside of them to display and section off data within the container.
  • In the mobile version of this container the width will shrink to span the width of the parent container.
  • Font size increases for mobile.
