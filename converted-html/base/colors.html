Color Palette

"Caution?" Color is also important when it comes to branding. The famous big yellow arch for McDonald's will always be yellow, not green or blue or purple.

Baseline

The following baseline colors are consistent with the Mercer brand and are used for all basic elements for applications and sites, like the base font and the navigation menu.

Example Style Hex Top Hex Bottom Usage Details
Solid #23334C -- Header Background - this is used as the background in the Main Navigation and is mostly viewable on the left and right sides of product header.
Gradient #F6F6F6 #CDCDCD Main Navigation - This background gradient is used on the main navigation bar in the Product Header.
Solid #002c77 -- Default Link - This color is used as the default link color.
Solid #333333 -- Page Headers - This color is used for all page headers or titles. See the Typography section for more details.
Solid #000000 -- Base Font - This color is used for the base font for all applications. See the Typography section for more details.

Confirmation/Positive

Green denotes a positive experience, and users tend to associate it with a confirmation, a positive result or a correct answer. In our applications, it is reserved for this use.

Example Style Hex Top Hex Bottom Usage Details
Solid #d6ebc3 -- Background - Commonly used as the background color on the Confirmation Message container.
Solid #2d7b05 -- Border - Used as the border color in the Confirmation Message container.
Solid #005804 -- Text - Confirmation/Positive - used as the text color when displaying a positive number, result or confirmation.

Example Style Hex Top Hex Bottom Usage Details

Attention/Highlight

The following colors are reserved for use with specific site/application entities (like Client profile, User profile, etc). The colors will be used primarily in the profile header for the specific entity. They will also be used as accents throughout various elements within the entity profile.

Example Style Hex Top Hex Bottom Usage Details
Solid #fef8e3 -- Background - Used in areas like the Attention Message container, as the hover color in the Interactive Data Table or as the background for the selected item in the Mini Filter.
Solid #c6a54f -- Border - Used as the border color in the Attention Message container.
Solid #8e5501 -- Text - Used as the text color for titles inside the Attention Message container, or in other areas as the text color to display highlighted text.

Interactive Data Tables

Interactive Data Tables need to have color so users can easily interpret the data presented, but at the same time, the colors should be soft as not to direct attention away from the table's content. Mercer's color scheme for all interactive data tables is a collection of soft light blues.

Example Style Hex Top Hex Bottom Usage Details
Solid #E9EEF5 -- Header Row - Background color for the top most row on a Interactive Data Table.
Solid #E8F0F7 -- Subheader Row - Background color used on the optional subheader row in a Interactive Data Table.
Solid #FFFFFF -- Alternating Data Row 1 - used on the Interactive Data Table in conjunction with Alternating Data Row 1 to create a "zebra-stripe" effect in the main content area to allow the user to easily identify a single row of data.
Solid #F7F7F7 -- Alternating Data Row 2 - used on the Interactive Data Table in conjunction with Alternating Data Row 1 to create a "zebra-stripe" effect in the main content area to allow the user to easily identify a single row of data.
Solid #EEEEEE -- Subtotal Row - Background color for the optional summary or total row that would appear as the last row in an Interactive Data Table.
Solid #E6E6E6 -- Total Row - Background color for the optional summary or total row that would appear as the last row in an Interactive Data Table.
Solid #F7F7F7 -- Sorting (odd row) -
Solid #EEEEEE -- Sorting (even row) -
Solid #A9BCD3 -- Table Header Border Color -

Buttons

Primary and secondary buttons can be easily distinguished by using different colors: green for primary and grey for secondary. The bright green will instantly draw users to primary buttons as they are the ones that need to be addressed first. There are slight visual variations for each of the four button states.

Example Style Hex Top Hex Bottom Usage Details
Gradient #69dd14 #2d6e0c This is the background color for the Primary Button (Default State).
Gradient #1b4a03 #69dd14 This is the background color for the Primary Button (Active State).
Gradient #69dd14 #1b4a03 This is the background color for the Primary Button (Hover State).
Solid #2D7B05 -- This is the background color for the Primary Button (Loading State).
Gradient #efefef #b0b0b0 This is the background color for the Secondary Button (Default State).
Gradient #9a9a9a #efefef This is the background color for the Secondary Button (Active State).
