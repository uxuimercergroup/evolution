The Static Data Table is an element meant to organize data on a page so it’s easy for the user to view and comprehend on a page or application.

Use the Static Data Table when you want to organize data that the user doesn’t need to interact with. If the user needs to interact with the data (search, filter, take clickable actions, etc) use the Interactive Data Table.

  • The Static Data Table works within the Mercer Grid System, which means the sizing is flexible and fluid to adapt to designer’s choices, user devices, and varying screen sizes.
  • Don’t use the Static Data Table in a Sidebar Container or Sidebar Layout, unless you only need to display a small amount of data.

Color

  • Coloring for each row in the Static Data Table will adhere to the Mercer Color Palette for all Data Tables.

Width

  • Designer can set column width using a helper class. Since the Static Data Table functions within the Mercer Grid System, it will automatically adapt and configure itself.

Row Details

  • Padding for Static Data Table rows are set to 5 pixels on the top and bottom and 10 pixels to the left and right.
  • Line height of Static Data Table rows are 13 pixels to keep text consistent with Mercer branding.
  • Some rows can be assigned a Right Align Helper Class to alter rows that might not have the same amount of content fields as other rows, such as the Subtotal and Total rows.
  • Text will wrap in basic rows to adapt to longer content, but the label rows will stay in the same position, which is vertically aligned to the top of the row.

When static and interactive data tables are viewed on mobile/tablet devices, the table will remain unchanged and displayed as seen on the desktop. When the user touches the screen (or swipes to the left) a scroll bar will appear which allows them to view the entire data table. They are then able to swipe right and left to view the contents. The Quick Filter appears at the top of the table and enables users to quickly find and select from a pre-populated list of values as they type.
