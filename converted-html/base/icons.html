Icons - Core Set

Icons can come in handy when you want to enhance the visual aspect of your application or just simply use a visual instead of plain text. In order for users to recognize an icon's meaning, the meaning must be universal. To keep all Mercer sites and applications consistent, we've listed the different Mercer icons used according to their respective functions.

Want to use an icon not found here? You can work with a Visual Designer to create a new icon that aligns to the Mercer brand and adheres to our visual standards. Contact Jeff Tudor for more information.

Icon Font

Our icons use an icon Web font based primarily on the IcoMoon library. The icons available in IcoMoon have been narrowed to this core set. These should cover most icon needs, but if you can't find something to meet your specific needs, you can access the additional set to see a complete list of available icons. We continually add new icons to the core set from the additional set as they are required.

Icon Positioning

Rules

Remember to put the icon code in the correct content order so the icon is before or after the label text

  • If you want an icon on the left, then have the icon HTML code before the label text inside the label tag
  • If you want an icon on the right, then have the icon HTML code after the label text inside the label tag
  • If you want an icon on the top, then have the icon HTML code before the label text inside the label tag, and add the icon top positioning class to the icon HTML code
  • If you want an icon on the bottom, then have the icon HTML code after the label text inside the label tag, and add the icon bottom positioning class to the icon HTML code
