The Content Modal Window provides information for the user that supplements the current page without taking the user away from it. This window allows large amounts of content to be displayed on a page in multiple layers and subsections without cluttering the user interface.

You can use the Content Modal Window when you need to provide more information for users without taking them away from the current page. For example, users need to read content on a homepage, but the designer wants to provide an extra layer of supplemental content they can opt to read to further comprehension. A Content Modal Window is the perfect solution.

Functionality

  • When the Content Modal Window is activated, the rest of the screen (the content/page behind it) is disabled until the user closes it by clicking on the X located in the top right corner or clicking on the overlay outside if the modal window.
  • Content Modal Windows will contain the "close" icon (white circle with a blue X inside) so the user can close the window when desired.

Width/Height

  • The Content Modal Window has two available widths: 980px and 1200px. This width is based on the overall width of the site underneath the modal window. If a site width of 980px is choosen, the content modal will be 980px wide.
  • The Content Modal Window's height is dependent on the height of the browser window.
  • Left hand column with "action" icons remains a fixed size
  • The content area becomes scrollable.
  • The overall font size increases.
  • "Close" scrolls with content area.

Cross browser compatibility

IE9 doesn't crop background gradients on containers with rounded corners, so we square the corners of every container.

Dependencies

The Content Modal Window is based off a jQuery UI control called Dialog Widget. For more information on the jQuery UI Dialog Widget, check out the jQuery UI Dialog Widget API. This site is maintained by the jQuery Foundation, not Mercer.

The Scroll Pane Plugin is dependent on jQuery as well.

Dependencies

  • jQuery 2.1.1
  • jQuery UI 1.11.2
