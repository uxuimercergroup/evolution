The Scroll Pane is an added control that allows excess information to be presented in a defined space. It essentially allows a container to have scrollable content while keeping the container fixed to a desired height.

Scroll Panes are best used to keep containers to a certain height no matter how much content there is on the page. Also use when the content section is expected to overflow the specified area allotted. Keep Scroll Panes vertical; avoid horizontal scrolling whenever possible.

Components

  1. Header Text
  2. Scroll Pane Window
  3. Scroll Bar

Height/Width

  • The Scroll Pane container's width and height placement can be adjusted accordingly, the default height is 300px.
  • Place the Scroll Bar on the right side of window (vertical scroll)
  • Place the Scroll Bar on the bottom of window (horizontal scroll) - Avoid whenever possible
  • Allow the Scroll Bar to appear by default or when the user mouses over the area

Visual Design

  • Design the Scroll Pane to fit preferred visual format.

In the mobile version, all text is slightly larger.
