The Advanced Filter is an optional filter that can be used when it is necessary to allow the user to manage the data set in more complex ways or to filter using data fields that might not necessarily be displayed on screen.…

The Advanced Filter modal should be used when:

  • It is important to allow the user to filter the data in complex ways (i.e., in ways other than column sorting, standard filtering via the Mini Filter or the Quick Filter.)
  • The user needs to be able to sort on data fields that do not appear in the table UI.

General

  • The Advanced Filer appears in the Modal Window control.

This component does not have a baseline mobile version as of this release. A responsive version will be coming soon.
