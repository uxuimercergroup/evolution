Snippet Shortcuts

The following Evolution snippet shortcuts are for use with the Sublime Text 2 text editor which is available for Windows or Mac, download Sublime Text 2. The shortcut key commands provided are only for use with the Windows version but can be easily changed for use on a Mac, just replace alt with ctrl when editing the key bindings.

Installation Instructions

(These instructions assume you are using Sublime Text 2 for Windows 7)

  1. Checkout evolution project from GitHub.
  2. Copy the files from inside your local evolution/snippets directory and paste them into the following directory on your Windows computer: C:\Users\{Your_Username}\AppData\Roaming\Sublime Text 2\Packages\User
  3. Open Sublime Text 2.
  4. Go to Preferences > Key Bindings - User, this will open a preferences document in Sublime.
  5. If you do not have any existing user key bindings then select all the code in this file and delete it. If you have existing user key bindings, insert the code from the next step between the "[ ]" in the document, making sure to seperate key binding calls with commas.
  6. Copy the user key binding code from the code box below: 
    //------------------------------------------------------------------------------------------

// DATE - June 13, 2014
// AUTHOR - Doug Fraize, Matthew Holmes, Vinod Bhatt
//------------------------------------------------------------------------------------------

[

	{"keys": ["alt+a"],			"command": "insert_snippet", "args":	{"name": "Packages/User/ui_box_advance.sublime-snippet"}},

Miscellaneous

Theming

Theming Instructions

Developers can theme Evolution at the site level by customizing the theme.css file located at the following location: assets/css/ or if using SASS customizing the theme_variables.scss file inside assets/css/partials/global.

Furthermore, if a developer wants to add custom theme styles or variables (if using SASS) to their project, those can be added to the bottom of theme.css or theme.scss (if using SASS) in the section called WEBSITE THEME STYLES. These files are located at assets/css/.

Theming Reference

If a developer is using SASS they can use the theme_variables.scss file in the Evolution core as a reference for the theming variables used in theme_variables.scss at the site level (assets/css/partials/global/theme_variables.scss). Developers who are using just CSS can use it as a CSS value reference. The file is located at assets/css/core/global/.
